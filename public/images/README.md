# Images

Place the following images in this folder:

- `hero-bg.jpg` — Hero section background (full-width landscape)
- `logo.png` — AYURCARE logo
- `retreat1.jpg` — Retreat card image 1
- `retreat2.jpg` — Retreat card image 2
- `yoga-bg.jpg` — Yoga section background
- `spa-bg.jpg` — Spa/treatment card image

All images are referenced as `/images/<filename>` from the frontend.
The server serves this folder via `express.static('public')`.
